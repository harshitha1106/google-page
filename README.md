# Google Page Clone

A responsive clone of the Google homepage built using HTML, CSS, and JavaScript. This project replicates the clean and minimal design of Google's search page.

## How to Use

1. Open the live demo or clone the repository.
2. The page will load with the Google homepage clone.
3. Type in the search bar and press Enter or click the search button.
4. Navigate through the responsive design on different screen sizes.

## Installation (Local Setup)

1. Clone the repository:
   ```bash
   git clone https://github.com/harshitha1106/google-page.git
   ```

2. Navigate to the project folder:
   ```bash
   cd google-page
   ```

3. Open `index.html` in your browser.
